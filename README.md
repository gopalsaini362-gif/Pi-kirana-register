# Pi Kirana Register - Pi Network Edition

एक आधुनिक कियाना दुकान प्रबंधन प्रणाली जो **Pi Network** के लिए डिज़ाइन की गई है।

## 🎯 मुख्य विशेषताएं

### 📦 **स्टॉक प्रबंधन**
- उत्पाद जोड़ें, संपादित करें, और हटाएं
- श्रेणी, इकाई, मात्रा, खरीद/बिक्री मूल्य प्रबंधन
- कम स्टॉक और स्टॉक से बाहर संकेतक
- खोज कार्यक्षमता

### 💳 **बिक्री प्रणाली**
- बहु-वस्तु कार्ट-आधारित बिक्रय
- प्रत्येक बिक्री के बाद स्वचालित स्टॉक कटौती
- **रीयल-टाइम Pi Network और USDT में बिल जनरेशन**
- CoinGecko API से लाइव विनिमय दरें
- प्रिंट और शेयर विकल्प

### 💰 **मुद्रा रूपांतरण**
- **INR** (भारतीय रुपया)
- **USDT** (टेदर)
- **Pi Network (π)**
- CoinGecko API से रीयल-टाइम दरें
- 5 मिनट की कैशिंग

### 📊 **खर्च ट्रैकिंग**
- किराया, बिजली, कर्मचारी आदि के साथ श्रेणीबद्ध खर्च
- आज के खर्च का कुल
- मासिक खर्च का कुल

### 📈 **सारांश डैशबोर्ड**
- फिल्टर: आज, सप्ताह, महीना, सभी समय
- बिक्री, लाभ, खर्च, शुद्ध आय के आंकड़े
- शीर्ष 5 विक्रय वस्तुएं
- स्टॉक मूल्यांकन

### ⚙️ **सेटिंग्स**
- दुकान का नाम अनुकूलन
- कम स्टॉक थ्रेसहोल्ड सेटिंग
- बिल मुद्रा चयन (USDT/Pi/INR)

### 📱 **डिज़ाइन**
- मोबाइल-पहली इंटरफेस
- डेस्कटॉप पर फोन-फ्रेम लेआउट
- गर्म नारंगी और एम्बर कियाना स्टोर थीम
- Lucide React आइकन

### 💾 **डेटा स्टोरेज**
- ऑफ़लाइन-पहली वास्तुकला
- localStorage में सभी डेटा
- इंटरनेट कनेक्शन के बिना काम करता है

## 🚀 शुरुआत करें

### आवश्यकताएं
- Node.js 18+
- npm या yarn

### स्थानीय स्थापना

```bash
# Repository क्लोन करें
git clone https://github.com/gopalsaini362-gif/Pi-kirana-register.git
cd Pi-kirana-register

# Dependencies स्थापित करें
npm install

# विकास सर्वर शुरू करें
npm run dev

# उत्पादन के लिए बनाएं
npm run build

# GitHub Pages पर तैनात करें
npm run deploy
```

### GitHub Pages पर लाइव

ऐप स्वचालित रूप से GitHub Pages पर तैनात किया जाता है:
👉 **https://gopalsaini362-gif.github.io/Pi-kirana-register**

## 💱 मुद्रा एकीकरण

### समर्थित मुद्राएं
- **₹ INR** - भारतीय रुपया (प्राथमिक)
- **USDT** - Tether (CoinGecko से)
- **π** - Pi Network (CoinGecko से)

### विनिमय दर स्रोत
- **API**: [CoinGecko](https://www.coingecko.com/en/api) (निःशुल्क, कोई प्रमाणीकरण नहीं)
- **अपडेट आवृत्ति**: 5 मिनट कैशिंग
- **फॉलबैक**: ऑफलाइन मोड में कैश्ड दरें

### बिल उदाहरण
```
₹500 = 6.02 USDT = 3.33 π
(Real-time CoinGecko rates)
```
